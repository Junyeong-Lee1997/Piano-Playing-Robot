#data_diatonic

직접 만든 데이터셋 (아이패드에 악보를 그려서 한 장에 28개의 코드 * 40장 = 총 1120개의 데이터셋 구축
이때 C Diatonic Chord로 C, Dm, Em, F, G7, Am, B, 빈칸으로 8개의 데이터셋 구축
총 8*1120=8960개의 데이터셋

#data_diatonic_after
악보의 특정 영역의 코드를 이미지 파일로 저장한 경로
총 8960개의 데이터셋을 그렸다.

#test_data
테스트를 진행할 악보들을 넣어놓은 경로

#test_data_set
테스트 진행할 악보에 대한 코드들을 순서대로 나열한 경로 폴더
(악보 하나당 총 56개의 데이터셋이 구축된다 => 마디당 2개 코드 총 28마디)

#Diatonic.csv
data_diatonic_after경로의 이미지 파일들을 28*28로 바꾸어 784개의 데이터 + 라벨링1개의 총 785개의 데이터가 코드 하나당 저장됨
총 8960개의 이미지가 있으므로
8960*785의 데이터셋으로 구성되어 있음.
(이 파일은 Custom_Dataset.ipynb 코드를 사용하여 만들 수 있음)

#Custom_Dataset.ipynb
학습 데이터셋 저장하고 csv 파일의 형태로 바꾸는 코드

#CNN_Handwriting_alphabet
